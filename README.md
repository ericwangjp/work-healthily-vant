# 「轻养随行」健康生活管理小程序 (Vant Weapp 精简版)

「轻养随行」是一款倡导健康生活的微信小程序原型。本项目基于经典的 **Vant Weapp 微信小程序组件库** 进行构建，包含了核心的登录流和首页功能模块，专注于轻量级、简便快速的项目骨架搭建和 Vant 组件库应用实践。

---

## 🌟 功能特性

- 🔑 **简约登录流**: 预置移动端用户登录与权限校验界面。
- 🏠 **极简主页**: 承载核心业务交互的基础框架，具有极佳的可扩展性。
- 📦 **轻量化依赖**: 核心基于 `@vant/weapp` 微信小程序组件，运行迅速、性能开销小。
- 🛠️ **模块化代码**: 页面结构清晰，适合作为二次开发的精简骨架项目。

## 🛠️ 安装与快速开始

### 1. 安装依赖
在项目根目录下，使用终端运行以下命令安装 Vant Weapp 等组件依赖：
```bash
npm install
```

### 2. 微信开发者工具配置
1. 打开 [微信开发者工具](https://mp.weixin.qq.com/debug/wxadoc/dev/devtools/download.html)，选择 **“导入项目”**。
2. 选择当前项目根目录 `work-healthily-vant`。
3. AppID 填写您自己的微信小程序 AppID（或使用测试号）。
4. 在开发者工具上方菜单选择：**工具 -> 构建 npm**（必须执行，以将 node_modules 构建为微信兼容的 `miniprogram_npm`）。
5. 确保开发者工具详情面板中，**“使用 npm 模块”** 已被勾选。
