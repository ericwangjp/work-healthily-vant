# 「轻养随行」V1.0 MVP 小版本 PRD（修正版）

## 一、 产品基础信息
- **产品名称**：「轻养随行」
- **产品形态**：微信小程序（V1.0 MVP版）
- **核心目标**：解决上班族办公 8 小时核心亚健康问题，提供碎片化、低门槛、隐秘化的基础健康干预方案
- **产品定位**：上班族碎片化亚健康基础管家

## 二、目标用户（精准聚焦，适配P0功能）

核心目标用户：22~42岁都市上班族，细分3类核心人群，聚焦其P0功能相关痛点，为后续扩展全场景功能预留用户基础

|人群分类|核心痛点（贴合P0功能）|P0功能适配需求|
|---|---|---|
|年轻白领（22-28岁）|颈椎酸痛、用眼疲劳、久坐不动|轻量拉伸、护眼放松、久坐提醒|
|加班中坚（29-38岁）|腰椎劳损、久坐不活动、忘记喝水|久坐提醒、腰椎放松动作、喝水提醒|
|资深职场人（39-42岁）|慢性肩颈痛、久坐不适、饮水量不足|肩颈舒缓动作、久坐提醒、喝水记录|

## 三、 P0 核心功能需求（修正版）

### 模块 1：智能定时提醒系统
- **功能描述**：根据用户设置的上班时间，推送微信服务消息提醒
- **交互逻辑**：
    - 首次进入引导设置「上班时间」（如 9:00-18:00）。
    - 默认提醒间隔：久坐 45min、护眼 60min、喝水 90min。
    - 基础自定义设置：支持关闭某类提醒、调整提醒间隔（30-120分钟可调），暂不添加“免打扰时段”（后续迭代补充）
    - 页面原型（简化，聚焦核心）：仅保留“提醒设置页”，包含标题、3类提醒开关、间隔调整滑块
- **提醒方式**：微信服务消息推送

### 模块 2：工位微运动库
- **功能描述**：满足「坐姿、小幅度、零器械、隐秘」的 1-5 分钟动作。
- **动作数量**：**修正为 17 个核心动作**。
    - 颈椎舒缓（6个）
    - 腰椎放松（4个）
    - 体态矫正（5个，新增「坐姿挺胸」、「肩胛骨内收」等）
    - 手腕放松（2个：手腕旋转、手腕拉伸）
- **交互逻辑**：首页入口 -> 运动库 -> 分类 -> 动作详情（图文） -> 完成打卡。动作完成后，弹出「完成打卡」按钮（一键点击，无繁琐填写，打卡仅记录次数，不统计时长）

### 模块 3：眼部健康专区
- **功能描述**：极简护眼动作，无需起身。
- **动作充实**：**从 2 个增加至 4 个**。
    - 远眺引导、眼球转动、**新增：眨眼放松、眼部按摩**。
- **交互逻辑**：**修正为拥有独立的动作详情页**（复用运动库详情页逻辑），提供更清晰的指引。

### 模块 4：喝水提醒
- **功能描述**：定时提醒喝水，记录饮水量。
- **修正后逻辑**：
    - **目标明确化**：每日目标默认为 **1500ml**。
    - **记录方式**：每次点击「已喝水」自动累加默认值（默认每次提醒喝200ml，可调整）
    - **达标反馈**：当日累计饮水量首次 ≥ 1500ml 时，弹出鼓励文案

## 四、 页面原型

- 进度区域实时同步「今日健康进度」
- 页面整体：背景色#FFFFFF，无多余装饰
